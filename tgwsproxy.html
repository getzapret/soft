Что это

Локальный прокси для Telegram Desktop

TG WS Proxy поднимает прокси-сервер прямо на твоём компьютере и заворачивает трафик Telegram Desktop в WebSocket-соединения — это помогает восстановить связь, когда провайдер режет обычное подключение к Telegram. Если WebSocket недоступен, программа автоматически переключается на прямое TCP-соединение. Никакие сторонние сервера не используются — весь прокси работает локально, на адресе 127.0.0.1.

Установка

Как подключить

  • 1
    Скачай и запусти TgWsProxy_windows_x64.exe — программа свернётся в трей.
  • 2
    Открой Telegram Desktop → Настройки → Продвинутые настройки → Тип подключения → Прокси.
  • 3
    Выбери тип SOCKS5, адрес 127.0.0.1, порт 1080.
  • 4
    Готово — Telegram Desktop подключится через локальный прокси.
Вопросы

Частое

Работает на телефоне? +
Нет, программа рассчитана только на Telegram Desktop — на компьютере. Для мобильного приложения нужны другие способы.
Заменяет ли это VPN? +
Нет. Прокси работает только для Telegram Desktop — браузер и остальные программы через него не пойдут.
Чем отличается от Zapret? +
Zapret — универсальный обход для разных сайтов и сервисов. TG WS Proxy заточен именно под Telegram Desktop и решает его специфичные проблемы с подключением.
TG WS Proxy не связан с Telegram FZ-LLC · Используй на свой страх и риск
